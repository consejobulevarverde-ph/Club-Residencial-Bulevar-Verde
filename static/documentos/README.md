# Documentos del Club Residencial Bulevar Verde

Esta carpeta contiene todos los documentos disponibles para los residentes del club.

## Estructura de Carpetas

```
documentos/
├── reglamentos/         # Reglamentos de convivencia, internos, uso de zonas
├── actas/              # Actas de asambleas
├── formularios/        # Formularios administrativos para residentes
├── comunicados/        # Comunicados oficiales de la administración
└── financiero/         # Presupuestos, estados financieros, informes
```

## Cómo agregar documentos

1. Coloca el archivo PDF en la carpeta correspondiente
2. El nombre del archivo debe ser descriptivo: `reglamento-convivencia.pdf`, `acta-2024-12.pdf`
3. Los documentos se mostrarán automáticamente en la página web

## Formato recomendado

- **Formato**: PDF
- **Nombre**: descriptivo, en minúsculas, con guiones (no espacios)
- **Ejemplo**: `solicitud-salon-social.pdf`, `presupuesto-2024.pdf`

## Nota

Los archivos en esta carpeta son de acceso público para todos los residentes del club.
