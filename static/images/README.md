# Imágenes

Coloca aquí las imágenes del club:
- Logo del club: `logo.png`
- Favicon: `favicon-16x16.png`, `favicon-32x32.png`
- Apple touch icon: `apple-touch-icon.png`
- Fotos de instalaciones
- Fotos de zonas comunes

## Formato recomendado
- Logo: PNG con fondo transparente
- Favicons: PNG en tamaños 16x16, 32x32, 180x180
- Fotos: JPG optimizadas para web
